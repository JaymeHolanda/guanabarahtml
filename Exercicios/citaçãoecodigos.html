Tags e Códigos

Tag Code

Essa tag serve para deixar monoespaçada o texto. Sendo excelente para apresentar códigos em JavaScript e outros.

print('Hello Word')

Porém o Tag Code não coloca a identação do código.

n = float(input('Qual o seu nome ?')) s = n + 2 if n > 10: print('Não quero :) ')

Usando a Tag pre

A tag pre. Serve justamente para a identação de códigos. Logo é necessário colocar ela antes da Tag code.

 
    
    
        n = float(input('Qual o seu nome ?'))
        s = n + 2
              if n > 10:
              print('Não quero :) ')

Listas

Imagine que você tem uma lista com mais de 1 00 pessoas. nessa Ordenamento:


1- Carlos
2-Gabriel
3- Luiz
Etc..

Imagine o trabalho que daria para modificar do 40 por exemplo. Teria que mudar toda ordem numérica.

Lista Simples com: Tag Order List (OL)

Deve ser usado OL externo e a lista deve ter a tag dentro LI

  1. Acordar Cedo
  2. Tomar Café da Manhã
  3. Passear com o cachorro
    4.

Dentro de OL, pode ser usado o type para escolher se a lista será numerica ou alfabética.

Exemplo:

  1. Acordar Cedo
  2. Tomar Café da manhça
  3. Passer com o cachorro
    4.

Caso queira começar apartir do 4 ou mais. Deve usar o Start depois do type

Exemplo:

  1. Acordar Cedo
  2. Tomar Café da manhça
  3. Passer com o cachorro
    4.
